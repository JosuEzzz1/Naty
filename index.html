Que es el desarrollo web

El desarrollo web es el proceso de creación y mantenimiento de sitios web, que incluye una variedad de tareas como la programación de códigos, el diseño visual, y la gestión de servidores, contenidos y bases de datos. No hay que confundir desarrollo web y diseño web.

El desarrollo web abarca la construcción de un sitio y sus componentes mediante lenguajes de programación y aplicaciones. Es importante tener en cuenta que se dispone de aproximadamente tres segundos para captar la atención de los usuarios a través de los elementos gráficos de la web, de ahí que el desarrollo web sea fundamental en los resultados que persiga un site.
